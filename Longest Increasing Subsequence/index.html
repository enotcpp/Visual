Longest Increasing Subsequence (LIS)

The goal is to find the length of the longest subsequence in a given array such that all elements of the subsequence are sorted in increasing order.

  • DP[i] stores the length of the LIS ending at index i.
  • For each element i, we look at all previous elements j < i.
  • If nums[i] > nums[j], we update DP[i] = max(DP[i], DP[j] + 1).
for i in range(len(nums)):
    for j in range(i):
        if nums[i] > nums[j]:
            dp[i] = max(dp[i], dp[j] + 1)
