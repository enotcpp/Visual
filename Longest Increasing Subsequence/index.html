DP ARRAY (LENGHTS)

Longest Increasing Subsequence

Для каждого элемента i мы опрашиваем всех его предшественников j. Если текущее число больше предыдущего, мы можем прибавить единицу к его результату.

Ключевая логика: dp[i] = max(dp[i], dp[j] + 1)

def lengthOfLIS(nums):
    n = len(nums)
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
