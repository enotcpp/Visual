Longest Increasing Subsequence

Each row i represents the search for the longest subsequence ending at that element. We look back at DP[j] values to find the maximum possible length to extend.

for i in range(len(nums)):
    for j in range(i):
        if nums[i] > nums[j]:
            dp[i] = max(dp[i], dp[j] + 1)
