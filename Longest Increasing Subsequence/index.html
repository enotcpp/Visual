Longest Increasing Subsequence: $O(N^2)$ visualizer.

LIS: Row-to-DP Mapping

In this view, each Row (i) in the matrix corresponds to the calculation of DP[i].

While moving through columns j, we compare nums[i] with nums[j]. The best result is then stored in the DP cell to the right of the row.

# Each i loop fills one dp[i]
for i in range(len(nums)):
    for j in range(i):
        if nums[i] > nums[j]:
            dp[i] = max(dp[i], dp[j] + 1)
for (int i = 0; i < n; i++) {
    for (int j = 0; j < i; j++) {
        if (nums[i] > nums[j])
            dp[i] = max(dp[i], dp[j] + 1);
    }
}
