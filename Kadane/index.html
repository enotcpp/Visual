Maximum Subarray Sum

At each element A[i], we decide: should we extend the current sum (cur + A[i]) or start a new subarray from (A[i]).

Viz note: BLUE pointer shows the start of the current accumulation.

def kadane(A):
    res = cur = A[0]
    for x in A[1:]:
        cur = max(x, cur + x)
        res = max(res, cur)
    return res
