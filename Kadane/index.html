Dynamic Programming Approach

We maintain an array dp[i] where each element stores the maximum sum of a subarray ending at index i.

def kadane(A):
    cur = res = A[0]
    for x in A[1:]:
        cur = max(x, cur + x)
        res = max(res, cur)
    return res
