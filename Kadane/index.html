Maximum Subarray Sum

Kadane's algorithm scans the array once. At each position, it decides: "Should I add the current element to my existing chain, or start a new chain from here?"

Formula: current_sum = max(x, current_sum + x)

def kadane(arr):
    max_so_far = float('-inf')
    current_max = 0
    
    for x in arr:
        current_max = max(x, current_max + x)
        max_so_far = max(max_so_far, current_max)
        
    return max_so_far
