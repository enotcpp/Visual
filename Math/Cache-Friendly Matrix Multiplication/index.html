Cache-Friendly Algorithm (i-k-j)

В отличие от классического метода (i-j-k), здесь мы меняем порядок вложенных циклов. Это позволяет обращаться к элементам матриц B и C последовательно, что идеально для кэша процессора.

  • Цикл i: фиксирует строку в A и C.
  • Цикл k: фиксирует множитель A[i][k] и строку в B.
  • Цикл j: линейно проходит по строкам, суммируя результат.
def multiply(A, B):
    rows_A, cols_A = len(A), len(A[0])
    cols_B = len(B[0])
    C = [[0] * cols_B for _ in range(rows_A)]

    for i in range(rows_A):
        for k in range(cols_A):
            factor = A[i][k]
            if factor == 0: continue
            for j in range(cols_B):
                C[i][j] += factor * B[k][j]
    return C
Matrix multiply(const Matrix& A, const Matrix& B) {
    int rows_A = A.size(), cols_A = A[0].size();
    int cols_B = B[0].size();
    Matrix C(rows_A, vector<long long>(cols_B, 0));

    for (int i = 0; i < rows_A; i++) {
        for (int k = 0; k < cols_A; k++) {
            if (A[i][k] == 0) continue; 
            for (int j = 0; j < cols_B; j++) {
                C[i][j] += A[i][k] * B[k][j];
            }
        }
    }
    return C;
}
