Binary Heap (Max-Heap)

A Binary Heap is a complete binary tree that maintains the heap property. In a Max-Heap, every parent node is greater than or equal to its children.

  • Insert: O(log N). Add to the end, then move up (sift up) to restore order.
  • Extract Max: O(log N). Replace root with the last element, remove the last, then move down (sift down) to restore order.
  • Storage: O(N). Efficiently stored in an array: children of node i are at 2i+1 and 2i+2.
#include <vector>
#include <algorithm>

void siftUp(std::vector<int>& heap, int i) {
    while (i > 0 && heap[(i - 1) / 2] < heap[i]) {
        std::swap(heap[i], heap[(i - 1) / 2]);
        i = (i - 1) / 2;
    }
}

void siftDown(std::vector<int>& heap, int i) {
    int maxIdx = i;
    int l = 2 * i + 1;
    int r = 2 * i + 2;
    if (l < heap.size() && heap[l] > heap[maxIdx]) maxIdx = l;
    if (r < heap.size() && heap[r] > heap[maxIdx]) maxIdx = r;
    if (i != maxIdx) {
        std::swap(heap[i], heap[maxIdx]);
        siftDown(heap, maxIdx);
    }
}
