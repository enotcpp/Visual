Binary Heap (Max-Heap)

Бинарная куча — это дерево, которое всегда заполняется слева направо. В Max-Heap корень всегда является максимальным элементом.

// Standard Max-Heap Insert
void push(int val) {
    heap.push_back(val);
    siftUp(heap.size() - 1);
}
