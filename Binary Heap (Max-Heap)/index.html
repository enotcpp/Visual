Массив (Array View):

Max-Heap — это дерево, где родитель всегда больше или равен своим детям.

  • Вставка: O(log n) — добавляем в конец и тянем вверх (Sift Up).
  • Удаление: O(log n) — заменяем корень последним и тянем вниз (Sift Down).
// Основная операция Sift Up
while (i > 0 && h[(i-1)/2] < h[i]) {
    swap(h[i], h[(i-1)/2]);
    i = (i-1)/2;
}
