This lab demonstrates how a Max-Heap works. Elements are stored in a linear array but represented as a complete binary tree.

  • Root: The maximum element (at index 0).
  • Parent: Located at (i-1)/2.
  • Children: Located at 2i+1 and 2i+2.
void siftUp(int i) {
    while (i > 0 && heap[(i-1)/2] < heap[i]) {
        swap(heap[i], heap[(i-1)/2]);
        i = (i-1)/2;
    }
}
