Max-Heap Logic

A binary heap is a complete binary tree. The Max-Heap property ensures that every parent node is greater than its children.

This allows us to find the maximum element in $O(1)$ and perform insertions or deletions in $O(\log n)$.

// Standard Max-Heap Bubble Up
void siftUp(int i) {
    while (i > 0 && heap[(i-1)/2] < heap[i]) {
        swap(heap[i], heap[(i-1)/2]);
        i = (i-1)/2;
    }
}
