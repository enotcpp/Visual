Input Array (nums)
LIS Tails (Minimal endings for each length)

This algorithm maintains an array tails, where tails[i] is the smallest tail of all increasing subsequences of length i+1.

  • For each x in nums, we find the first element in tails that is ≥ x using Binary Search.
  • If such an element exists, we replace it with x (improving the "potential" for future elements).
  • If x is larger than all elements in tails, we append it (increasing the LIS length).
