Input Array (nums)
LIS Tails Array (O(n log n) state)

Binary Search Approach (O(N log N))

This algorithm maintains an array tails, where tails[i] stores the smallest tail of all increasing subsequences of length i+1.

For each element n in nums, we use Binary Search (`lower_bound`) to find the first element in tails that is ≥ n.

  • tails.push_back(n): If n is larger than all existing tails, the LIS length increases.
  • *it = n: If we find a tail ≥ n, we replace it. This doesn't change LIS length, but makes the "foundation" smaller for future numbers.
int lengthOfLIS(vector<int>& nums) {
    if (nums.empty()) return 0;
    vector<int> tails;
    for (int n : nums) {
        // Binary search for the first element >= n
        auto it = lower_bound(tails.begin(), tails.end(), n);
        if (it == tails.end()) {
            tails.push_back(n); // Extend LIS
        } else {
            *it = n; // Improve existing tail
        }
    }
    return tails.size();
}
