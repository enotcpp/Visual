Binary Search Approach

Instead of comparing everything, we keep track of the best possible increasing subsequences of each length.

  • tails[i] is the smallest possible end element of an increasing subsequence of length i+1.
  • When we see a new number n, we use lower_bound to find the first tail ≥ n.
  • We replace it to keep the tail as small as possible (making it easier to attach future numbers).
int lengthOfLIS(vector<int>& nums) {
    vector<int> tails;
    for (int n : nums) {
        // Binary search for the first element >= n
        auto it = lower_bound(tails.begin(), tails.end(), n);
        
        if (it == tails.end()) {
            tails.push_back(n); // Extend LIS
        } else {
            *it = n; // Improve existing tail
        }
    }
    return tails.size();
}
