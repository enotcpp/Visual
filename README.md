# 📚 AlgoView

[English](#english) | [Deutsch](#deutsch)

## English

### Project Overview
**AlgoView** is an open-source educational platform dedicated to the interactive visualization of classic computer science algorithms. The goal of this project is to transform complex logical structures and mathematical steps into clear, step-by-step visual experiences.

### Goals
* **Educational Clarity:** Help students and developers understand how algorithms work under the hood.
* **Interactive Learning:** Allow users to input their own data and trace the execution at their own pace.
* **Open Source:** Provide clean, accessible code for anyone to learn from, modify, or contribute to.

### Features
* Step-by-step execution with "Back" and "Next" controls.
* Real-time pointer and index tracking ($i$, $j$, $pivot$, etc.).
* Visual highlighting of modified data ranges.
* Responsive and modern Dark Mode interface.

## Deutsch

### Projektübersicht
**AlgoView** ist eine Open-Source-Bildungsplattform, die sich der interaktiven Visualisierung klassischer Informatik-Algorithmen widmet. Das Ziel dieses Projekts ist es, komplexe logische Strukturen und mathematische Abläufe in klare, schrittweise visuelle Erlebnisse zu verwandeln.

### Ziele
* **Didaktische Klarheit:** Unterstützung für Studierende und Entwickler beim Verständnis der internen Funktionsweise von Algorithmen.
* **Interaktives Lernen:** Benutzer können eigene Daten eingeben und die Ausführung im eigenen Tempo verfolgen.
* **Open Source:** Bereitstellung von sauberem, zugänglichem Code für alle, die lernen, modifizieren oder zum Projekt beitragen möchten.

### Funktionen
* Schritt-für-Schritt-Ausführung mit „Zurück“- und „Weiter“-Steuerung.
* Echtzeit-Tracking von Zeigern und Indizes ($i$, $j$, $pivot$ usw.).
* Visuelle Hervorhebung von geänderten Datenbereichen.
* Moderne Benutzeroberfläche im Dark Mode.
