This visualization uses the Contribution Technique to solve the "Arithmetic Slices" problem. For any new element nums[i], if it forms an arithmetic progression with its predecessors, it contributes exactly one more valid slice than the contribution of the previous element. The total count is simply the sum of all individual contributions.
def numberOfArithmeticSlices(nums): total, current_contrib = 0, 0 for i in range(2, len(nums)): if nums[i] - nums[i-1] == nums[i-1] - nums[i-2]: current_contrib += 1 total += current_contrib else: current_contrib = 0 return total
