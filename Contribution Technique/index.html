Array Processing:
Subarrays (Arithmetic Slices):

Each new element A[i] adds (contribution) new slices. If A[i]-A[i-1] == A[i-1]-A[i-2], the new contribution is prev_contribution + 1. Otherwise, it resets to 0.

def numberOfArithmeticSlices(nums): total, current_contrib = 0, 0 for i in range(2, len(nums)): if nums[i] - nums[i-1] == nums[i-1] - nums[i-2]: current_contrib += 1 total += current_contrib else: current_contrib = 0 return total
int numberOfArithmeticSlices(vector<int>& nums) { int total = 0, current_contrib = 0; for (int i = 2; i < nums.size(); i++) { if (nums[i]-nums[i-1] == nums[i-1]-nums[i-2]) { current_contrib++; total += current_contrib; } else current_contrib = 0; } return total; }
