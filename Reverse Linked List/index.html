How Reversal Works

Чтобы развернуть список, мы проходим по нему и для каждого узла меняем его поле next так, чтобы оно указывало на предыдущий узел.

  • Save next: Сначала сохраняем адрес следующего узла, иначе мы потеряем доступ к остальной части списка.
  • Link back: Перенаправляем текущий узел на prev.
  • Shift: Сдвигаем prev и curr на один шаг вперед.
ListNode* reverseList(ListNode* head) {
    ListNode *prev = nullptr, *curr = head;
    while (curr) {
        ListNode* next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
