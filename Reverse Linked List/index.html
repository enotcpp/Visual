Algorithm Theory

Для разворота односвязного списка мы используем итеративный подход с тремя указателями:

  • curr: текущий узел для обработки.
  • prev: узел, на который мы перенаправим связь (предыдущий).
  • next: временный указатель для сохранения ссылки на остаток списка.

Сложность по времени: O(n). Сложность по памяти: O(1).

ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr;
    ListNode* curr = head;
    while (curr) {
        ListNode* next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
