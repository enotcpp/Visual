Reverse Linked List Algorithm

Чтобы развернуть односвязный список, необходимо пройти по нему и перенаправить указатель next каждого узла на предыдущий.

  • Save: Сначала сохраняем адрес следующего узла, чтобы не потерять список.
  • Re-link: Перенаправляем текущий узел назад.
  • Shift: Сдвигаем prev и curr вперед.
ListNode* reverseList(ListNode* head) {
    ListNode *prev = nullptr;
    ListNode *curr = head;
    while (curr) {
        ListNode* next = curr->next;
        curr->next = prev; // Связь теперь указывает назад
        prev = curr;
        curr = next;
    }
    return prev;
}
