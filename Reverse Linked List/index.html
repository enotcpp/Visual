Reverse Linked List

Алгоритм разворота списка перенаправляет указатель `next` каждого узла на предыдущий элемент.

  • prev: Изначально `nullptr`, в конце станет новой головой.
  • curr: Текущий узел, который мы обрабатываем.
  • next: Временный указатель на "будущее", чтобы не потерять хвост списка при разрыве связи.
ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr;
    ListNode* curr = head;
    while (curr != nullptr) {
        ListNode* next = curr->next; // Сохраняем ссылку на следующий
        curr->next = prev;           // Разворачиваем текущий узел
        prev = curr;                 // Двигаем prev вперед
        curr = next;                 // Двигаем curr вперед
    }
    return prev;
}
