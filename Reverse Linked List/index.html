Linked List Reversal

Итеративный разворот списка требует перенаправления каждого указателя на предыдущий элемент.

  • prev: Сосед слева (изначально nullptr).
  • curr: Узел, который мы разворачиваем в данный момент.
  • next: Временная ссылка, чтобы не потерять хвост списка.
ListNode* reverseList(ListNode* head) {
    ListNode *prev = nullptr, *curr = head;
    while (curr) {
        ListNode* next = curr->next;
        curr->next = prev; // Ключевой шаг
        prev = curr;
        curr = next;
    }
    return prev;
}
