Reverse Linked List

Алгоритм разворота списка — это классическая задача на манипуляцию указателями. Мы проходим по списку один раз, меняя направление каждой связи.

ListNode* reverseList(ListNode* head) {
    ListNode *prev = nullptr, *curr = head;
    while (curr != nullptr) {
        ListNode* next = curr->next; // SAVE
        curr->next = prev;           // REVERSE
        prev = curr;                 // MOVE PREV
        curr = next;                 // MOVE CURR
    }
    return prev;
}
