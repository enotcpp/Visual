Reverse Linked List

Задача: Развернуть односвязный список. Узел `1 -> 2 -> 3` должен стать `3 -> 2 -> 1`.

Алгоритм: Мы используем три указателя, чтобы не потерять список в процессе перестроения связей:

  • `curr`: Текущий узел, который мы обрабатываем.
  • `prev`: Узел, который станет новым "следующим" для `curr`.
  • `next`: Временный указатель, чтобы сохранить ссылку на остальную часть списка перед тем, как разорвать связь у `curr`.
Node* reverseList(Node* head) {
    Node* prev = nullptr; // Новым хвостом станет null
    Node* curr = head;
    
    while (curr != nullptr) {
        Node* nextTemp = curr->next; // 1. Сохраняем "будущее"
        curr->next = prev;         // 2. РАЗВОРАЧИВАЕМ стрелку
        prev = curr;               // 3. Сдвигаем prev вперед
        curr = nextTemp;           // 4. Сдвигаем curr вперед
    }
    return prev; // prev теперь указывает на новую голову
}
