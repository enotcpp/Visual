Reverse Linked List

Алгоритм разворота списка работает путем перенаправления указателя next каждого узла на предыдущий элемент.

  • Save: Временно сохраняем curr->next.
  • Link: Разворачиваем стрелку: curr->next = prev.
  • Shift: Сдвигаем prev и curr вперед.
ListNode* reverseList(ListNode* head) {
    ListNode *prev = nullptr, *curr = head;
    while (curr) {
        ListNode* nextTemp = curr->next;
        curr->next = prev; // Разворот
        prev = curr;
        curr = nextTemp;
    }
    return prev;
}
