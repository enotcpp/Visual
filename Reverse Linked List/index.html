Reverse Linked List Algorithm

Задача — изменить направление всех указателей в списке на противоположное за один проход.

  • Time Complexity: O(n) — посещаем каждый узел один раз.
  • Space Complexity: O(1) — используем только три дополнительных указателя.
  • Strategy: На каждом шаге мы сохраняем ссылку на следующий узел, чтобы не потерять его, затем разворачиваем текущий узел назад.
ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr;
    ListNode* curr = head;
    while (curr != nullptr) {
        ListNode* nextTemp = curr->next; // 1. Сохраняем
        curr->next = prev;               // 2. Разворачиваем
        prev = curr;                     // 3. Сдвигаем prev
        curr = nextTemp;                 // 4. Сдвигаем curr
    }
    return prev;
}
